Data Science Foundation Program FAQs

How Does DSFP (Data Science foundation program ) Works?

It follows a rigorous certification process to improve the Data Literacy. It is based on workshop approach where learner has to sticked with the approach and do the handson and develop multiple project by themselves once they complete program then they would be awarded with Data Science Foundation Program certificate.

What are the prerequisite for this Data Science Foundation Programme?

No prerequisite required. Python basic programming knowledge is added advantage and if you don't know then don't worry we would teach the Python Basic Programming in the begining of the course.

Who should take up this Data Science online course?

Anyone who is interested to play with data and want to take decision depending upon data.

How to attend this online program?

This is an online program and not a classroom program. You need to have a laptop and a stable internet connection with a minimum speed of 5Mbps with headphone and attend at any place.

What are the System Requirements for this program?

The basic system requirements:

  • Windows10 64 bit operating system
  • A minimum 4GB RAM required but 8GB is recommendable
  • A hedaphone with microphone

We request you to use the personal laptop rather than office laptop to avoid the installation issues.

Why this program is important for you?

This program would give the foundation knowledge for Data Science from where you can take the decision based on the data.

What if you miss a class?

The live session would be recorded and post 6 hours of the online session, you would get the link to download the video and practice the exercise. But, we request you to not miss any individual session.

Will i get any assistance if refer anyone?

You would get a gift voucher worth 500INR. Your friend should continue the course then only you would get this assistance. If one drops then you won't be eligible.
